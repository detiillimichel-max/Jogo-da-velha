🎮 Jogo da Velha Infantil
Um jogo da velha colorido e divertido feito para crianças! 😸🐶
✨ Funcionalidades
😸 Gatinho vs 🐶 Cachorro — sem X e O chatos, só emojis!
🎊 Confete explode quando alguém ganha
🔊 Sons suaves a cada jogada
📊 Placar salvo durante a sessão
📱 Totalmente responsivo — funciona no celular e no computador
🌈 Bolinhas coloridas animadas no fundo
🚀 Como jogar
Acesse o jogo pelo link do GitHub Pages
O Gatinho 😸 sempre começa
Clique em qualquer quadradinho para jogar
Quem completar 3 em linha vence! 🏆
Clique em 🔄 Nova Partida para jogar de novo
Clique em 🗑️ Zerar Placar para resetar tudo
🛠️ Tecnologias
HTML5
CSS3 (animações, variáveis CSS, flexbox, grid)
JavaScript puro (sem frameworks)
Web Audio API (sons sem arquivos externos)
Google Fonts — Fredoka One + Nunito
📁 Estrutura
Código
🌐 Deploy no GitHub Pages
Faça o fork ou crie um repositório
Coloque o index.html na raiz
Vá em Settings → Pages
Em Source, selecione main / root
Pronto! O jogo estará disponível em:
https://seu-usuario.github.io/nome-do-repo
